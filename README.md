# 🎯 Customer Churn Prediction
### UAS Bengkel Koding Data Science — UDINUS

## 📌 Deskripsi Project
Proyek ini bertujuan membangun model prediksi churn pelanggan menggunakan
dataset Sales and Marketing Customer (15.000 records).

## 📁 Struktur Project
- `data/` — Dataset mentah
- `notebook/` — Jupyter Notebook analisis & modeling
- `model/` — Model terbaik hasil training
- `app/` — Aplikasi deployment Streamlit

## 📊 Dataset
- Sumber: [Kaggle - Sales and Marketing Dataset](https://www.kaggle.com/datasets/bhaskerpaul/sales-and-marketing-dataset)
- Jumlah data: 15.000 records, 30 kolom
- Target: `churn` (0 = tidak churn, 1 = churn)

pandas, numpy, matplotlib, seaborn, scikit-learn, streamlit
